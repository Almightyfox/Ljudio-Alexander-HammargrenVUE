# Ljudio
_MusicPlayer project_

## Technologies:

```text
Vue 3 vite
Visual studio code
```

## Project setup

- Clone the repo
- Ensure terminal is looking at the correct folder (the folder of this project or a folder of your choice!)
- Run the following

```text
npm install
npm run dev
```

- Default frontend localhost:3000
